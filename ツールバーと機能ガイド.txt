  1. ツールバー (画面上部のアイコン列)
================================================================================

左から順に配置されています。アイコンの上にマウスを置くとツールチップが出ます。

▼ ファイル操作
  ┌──────────────────────────────────────────────────────────────────┐
  │ [新規]                                                            │
  │   新しい空のプロジェクトを作成します。                             │
  │   解像度 (1920x1080 など)、フレームレート (30fps / 60fps など)、   │
  │   プロジェクト名を設定するダイアログが開きます。                   │
  │   ショートカット: Ctrl + N                                         │
  └──────────────────────────────────────────────────────────────────┘
  ┌──────────────────────────────────────────────────────────────────┐
  │ [開く]                                                            │
  │   動画ファイルを開きます。                                         │
  │   対応形式: MP4, MKV, MOV, WebM, FLV                               │
  │   開いたファイルは自動的にタイムラインに配置されます。             │
  │   ショートカット: Ctrl + O                                         │
  └──────────────────────────────────────────────────────────────────┘
  ┌──────────────────────────────────────────────────────────────────┐
  │ [保存]                                                            │
  │   現在のプロジェクトを .veditor ファイルに保存します。             │
  │   タイムラインのクリップ配置、エフェクト、設定がすべて保存されます。│
  │   ショートカット: Ctrl + S                                         │
  └──────────────────────────────────────────────────────────────────┘

▼ 編集操作
  ┌──────────────────────────────────────────────────────────────────┐
  │ [元に戻す]                                                        │
  │   直前の操作を取り消します (カット、削除、速度変更など)。          │
  │   ショートカット: Ctrl + Z                                         │
  └──────────────────────────────────────────────────────────────────┘
  ┌──────────────────────────────────────────────────────────────────┐
  │ [やり直し]                                                        │
  │   元に戻した操作をもう一度実行します。                             │
  │   ショートカット: Ctrl + Shift + Z                                 │
  └──────────────────────────────────────────────────────────────────┘

▼ クリップ編集
  ┌──────────────────────────────────────────────────────────────────┐
  │ [分割]                                                            │
  │   現在の再生ヘッド位置でクリップを 2 つに分割します。              │
  │   不要部分の切り取りや、クリップの中間にエフェクトを差し込む時に  │
  │   使用します。                                                     │
  │   ショートカット: S                                                │
  └──────────────────────────────────────────────────────────────────┘
  ┌──────────────────────────────────────────────────────────────────┐
  │ [削除]                                                            │
  │   選択中のクリップをタイムラインから削除します。                   │
  │   ショートカット: Delete                                           │
  │                                                                    │
  │   リップル削除 (削除後に後続クリップを詰める):                     │
  │   ショートカット: Shift + Delete                                   │
  └──────────────────────────────────────────────────────────────────┘
  ┌──────────────────────────────────────────────────────────────────┐
  │ [コピー]                                                          │
  │   選択中のクリップをクリップボードに複製します。                   │
  │   ショートカット: Ctrl + C                                         │
  └──────────────────────────────────────────────────────────────────┘
  ┌──────────────────────────────────────────────────────────────────┐
  │ [貼付]                                                            │
  │   クリップボードのクリップを現在の位置に貼り付けます。             │
  │   ショートカット: Ctrl + V                                         │
  └──────────────────────────────────────────────────────────────────┘

▼ エフェクト・装飾
  ┌──────────────────────────────────────────────────────────────────┐
  │ [テキスト]                                                        │
  │   テロップ・字幕・タイトルを追加します。                           │
  │   フォント、色、アウトライン、影、アニメーションを設定できます。   │
  │   ショートカット: T                                                │
  └──────────────────────────────────────────────────────────────────┘
  ┌──────────────────────────────────────────────────────────────────┐
  │ [色補正]                                                          │
  │   明るさ、コントラスト、彩度、色相、露出、ホワイトバランス、       │
  │   ハイライト/シャドウ、ガンマを調整します。変更はリアルタイムで    │
  │   GPU シェーダーによってプレビューに反映されます。                 │
  │   ショートカット: Ctrl + G                                         │
  └──────────────────────────────────────────────────────────────────┘
  ┌──────────────────────────────────────────────────────────────────┐
  │ [効果]                                                            │
  │   ビデオエフェクト (ブラー、シャープ、モザイク、ビネットなど) を   │
  │   選択クリップに適用します。                                       │
  │   ショートカット: Ctrl + Shift + F                                 │
  └──────────────────────────────────────────────────────────────────┘
  ┌──────────────────────────────────────────────────────────────────┐
  │ [マーカー]                                                        │
  │   現在の再生ヘッド位置にマーカーを付けます。チャプター作成や       │
  │   後で戻ってきたい場所の目印として使います。                       │
  │   ショートカット: Ctrl + M                                         │
  └──────────────────────────────────────────────────────────────────┘

▼ 出力・録画
  ┌──────────────────────────────────────────────────────────────────┐
  │ [出力]                                                            │
  │   タイムラインの内容を動画ファイルとしてエクスポートします。       │
  │   コーデック (H.264 / H.265 / AV1 など)、ビットレート、解像度、    │
  │   FPS、音声コーデックなどを設定できます。                          │
  │   ショートカット: Ctrl + E                                         │
  └──────────────────────────────────────────────────────────────────┘
  ┌──────────────────────────────────────────────────────────────────┐
  │ [録画]                                                            │
  │   画面録画を開始します。録画した動画は自動でタイムラインに         │
  │   追加されます。停止は Tools メニューから。                        │
  └──────────────────────────────────────────────────────────────────┘


================================================================================
  2. メニューバー
================================================================================

--------------------------------------------------------------------------------
  2-1. ファイル メニュー
--------------------------------------------------------------------------------

  ● 新規プロジェクト (Ctrl+N)
    新しいプロジェクトを作成します。最初に解像度とフレームレートを決めます。

  ● ファイルを開く (Ctrl+O)
    動画ファイルを直接読み込み、タイムラインに追加します。

  ● 最近使ったファイル
    過去に開いた動画・プロジェクトの一覧から選んで再度開けます。

  ● プロジェクトを開く (Ctrl+Shift+O)
    .veditor プロジェクトファイルを開きます。

  ● プロジェクトを保存 (Ctrl+S)
    現在のプロジェクトを上書き保存します。

  ● 名前を付けて保存 (Ctrl+Shift+S)
    別名で保存します。

  ● エクスポート (Ctrl+E)
    動画ファイルとして書き出します (コーデック・ビットレート指定)。

  ● Remotion形式でエクスポート
    Remotion (React ベースの動画生成フレームワーク) のプロジェクト形式で
    出力します。Web 上で動的な動画を生成したい場合に便利です。

  ● 終了 (Ctrl+Q)
    アプリケーションを終了します。


--------------------------------------------------------------------------------
  2-2. 編集 メニュー
--------------------------------------------------------------------------------

  ● 元に戻す (Ctrl+Z) / やり直す (Ctrl+Shift+Z)
  ● クリップをコピー (Ctrl+C) / 貼り付け (Ctrl+V)
  ● 再生ヘッドで分割 (S)
  ● クリップを削除 (Delete) / リップル削除 (Shift+Delete)
  ● スナップ切替 (N)
    クリップ境界にピッタリ吸着するかどうかを切り替えます。細かい調整を
    したい時はオフ、綺麗に並べたい時はオンにします。
  ● 再生速度を設定
    選択クリップの再生速度を 0.25x ～ 4.0x の間で指定します。
  ● キーボードショートカット (Ctrl+Alt+K)
    ショートカットのカスタマイズダイアログを開きます。


--------------------------------------------------------------------------------
  2-3. 表示 メニュー
--------------------------------------------------------------------------------

  ● 拡大 / 縮小 (Ctrl+= / Ctrl+-)
    タイムラインのズームを変更します。細かい編集時は拡大、全体を
    俯瞰したい時は縮小。
  ● テーマ変更
    Dark / Light / Midnight / Ocean の 4 種から選択できます。
  ● ツールバーのツールチップを表示
  ● ツールバーをアイコンのみ表示


--------------------------------------------------------------------------------
  2-4. トラック メニュー
--------------------------------------------------------------------------------

  ● ビデオトラックを追加
    V1, V2, V3... と複数の映像トラックを重ねられます。上のトラックが
    下のトラックを覆います。PiP や合成に使います。
  ● オーディオトラックを追加
    BGM、ナレーション、効果音などをそれぞれ別トラックで扱えます。


--------------------------------------------------------------------------------
  2-5. 挿入 メニュー
--------------------------------------------------------------------------------

  ● テキスト / テロップ追加 (T)
    新しいテキストオーバーレイをクリップに追加。
  ● テキスト管理 (Ctrl+T)
    既存テキストオーバーレイの一覧を確認・編集。
  ● 字幕をインポート (SRT/VTT)
    外部の字幕ファイルを読み込んでテキストオーバーレイとして適用。
  ● テキストテンプレートを保存
    よく使うスタイルをテンプレート化。
  ● トランジションを追加
    クリップ間にフェード、ワイプなどのトランジションを挿入。
  ● 画像 / 静止画を追加
    PNG / JPEG などを配置。
  ● ピクチャー・イン・ピクチャー追加
    小窓で別クリップを重ねる PiP 機能。


--------------------------------------------------------------------------------
  2-6. オーディオ メニュー
--------------------------------------------------------------------------------

  ● 音量を設定
    選択クリップの音量を 0% (ミュート) ～ 200% (増幅) で指定。
  ● BGM / 音声ファイルを追加
    別の音声トラックに BGM を乗せます。
  ● ミュート切替 (M) / ソロ切替
    A1 トラックを一時的に消す / そのトラックだけ聞く。
  ● イコライザー
    プリセット (ボーカル強調・低音強調・ラジオ風など) で音質を調整。
  ● オーディオエフェクト
    リバーブ、コンプレッサー、ノーマライズ、フェードイン/アウト、
    バスブースト、ボイスエンハンスを適用。
  ● VST / AUプラグイン
    外部プラグインを読み込んでエフェクトチェーンに追加。


--------------------------------------------------------------------------------
  2-7. マーカー メニュー
--------------------------------------------------------------------------------

  ● 再生ヘッドにマーカー追加 (Ctrl+M)
    現在の位置にマーカー。種類は Standard / Chapter / Todo / Note。
  ● 全マーカーを表示
    マーカーの一覧を確認。
  ● YouTubeチャプターをエクスポート
    Chapter 種別のマーカーから YouTube の説明欄用チャプターテキストを
    生成します。


--------------------------------------------------------------------------------
  2-8. エフェクト メニュー
--------------------------------------------------------------------------------

  ● 色補正 / グレーディング (Ctrl+G)
    GPU シェーダーで高速に色を調整。プレビューにリアルタイム反映。
  ● ビデオエフェクト (Ctrl+Shift+F)
    ブラー / シャープ / モザイクなどの映像エフェクト。
  ● プラグインエフェクト
    自作・サードパーティ製プラグインを適用。
  ● LUT適用 (.cube)
    Film 風 LUT を適用して色味を変えられます。
  ● LUT管理
    登録済み LUT 一覧を確認。
  ● エフェクトプリセット適用 (Ctrl+P)
    事前に作ったエフェクトの組み合わせを 1 クリックで適用。
  ● 現在設定をプリセットに保存
    使い回したい色補正 + エフェクトの組合わせを保存。
  ● プリセット管理
    保存済みプリセットの一覧と削除。
  ● キーフレーム編集 (Ctrl+K)
    色補正値を時間経過でアニメーションさせます。
  ● GPUシェーダーエフェクト
    GLSL シェーダーによる高度な映像加工。
  ● GPUシェーダー管理
    シェーダーライブラリを確認。


--------------------------------------------------------------------------------
  2-9. 再生 メニュー
--------------------------------------------------------------------------------

  ● J / K / L 速度コントロール
    - J: 逆再生 / 速度を下げる
    - K: 再生 / 一時停止
    - L: 順再生 / 速度を上げる
    L を連打するたびに 2x, 4x, 8x と速度が上がります。
  ● イン点をマーク (I) / アウト点をマーク (O)
    再生範囲や書き出し範囲を指定します。


--------------------------------------------------------------------------------
  2-10. ツール メニュー (AI・自動編集)
--------------------------------------------------------------------------------

  ● 無音検出
    音声の無音区間を自動検出。
  ● 自動ジャンプカット
    無音区間をまとめて削除する "ジェットカット" を実行。
  ● シーン変化検出
    映像のシーン切り替え位置を自動検出。

  ● 手ブレ補正
    ハンドヘルド動画を 2-pass で安定化。

  ● スピードランプ (可変速)
    時間経過で速度を変える (例: 1x → 2x に加速)。

  ● モーショントラッキング
    指定領域を追跡し、オーバーレイを追従させます。

  ● 音声ノイズ除去 / 映像ノイズ除去
    ホワイトノイズやフィルムグレインを除去。

  ● 字幕自動生成 (Whisper)
    OpenAI Whisper で音声から字幕を自動生成します。
    (要: pip install openai-whisper)

  ● AI自動ハイライト
    映像のダイジェスト動画を自動生成します。

  ● 画面録画を開始 / 停止
    画面キャプチャ機能。

  ● プロキシモード切替 / プロキシ生成
    4K / 8K 動画を軽量な低解像度プロキシに置き換えて編集を軽くします。

  ● レンダーキュー (Ctrl+Shift+R)
    複数のエクスポートをキューに入れて一括実行。

  ● ネットワークレンダー
    LAN 内の他のマシンを使って分散レンダリング。

  ● Pythonスクリプトコンソール (Ctrl+Shift+P)
    Python スクリプトでプロジェクトを操作できます。

  ● マルチカメラセットアップ / 切替
    複数カメラ素材を同期してカメラ切替編集ができます。


--------------------------------------------------------------------------------
  2-11. コンポジション メニュー (After Effects 風機能)
--------------------------------------------------------------------------------

  ● シェイプレイヤー追加
    長方形 / 楕円 / 星形 / ポリゴンなどを配置。
  ● パーティクルエフェクト追加
    雪・火花・煙などのプリセット。
  ● テキストアニメーション追加
    タイプライター / フェード / スライドなどのテキストアニメ。
  ● トランスフォームキーフレーム編集
    位置・スケール・回転・不透明度を時間軸でアニメ。
  ● マスク追加
    矩形 / 楕円 / ポリゴン マスクで一部だけを表示。
  ● ワープ / 歪みエフェクト
    メッシュワープ / パペットピン / バルジ / ピンチなど。
  ● エクスプレッション
    プロパティを式で制御 (例: wiggle(2, 10))。
  ● 選択をプリコンポーズ
    選択したクリップをひとつのコンポジションにまとめます。


--------------------------------------------------------------------------------
  2-12. ヘルプ メニュー
--------------------------------------------------------------------------------

  ● 無料素材ガイド (F1)
    ロイヤリティフリー素材サイトへのリンク集。
  ● バージョン情報
    アプリのバージョンを表示。

  4. よくある操作の手順
================================================================================

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
  Q1. 動画をカットして不要な部分を削除したい
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

   1. [開く] で動画をタイムラインに配置
   2. タイムライン下部のシークバーで不要部分の開始位置へ
   3. S キー (または [分割] ボタン) でカット
   4. 不要部分の終了位置へ移動し、再度 S キーでカット
   5. 中央の不要クリップを選択して Delete
   6. Shift + Delete なら後ろのクリップが前に詰まる (リップル削除)


━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
  Q2. テロップ (字幕) を入れたい
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

   1. テロップを入れたい位置へ再生ヘッドを移動
   2. クリップを選択
   3. T キー (または [テキスト] ボタン) を押す
   4. ダイアログで文字・フォント・色・アウトライン・位置を設定
   5. OK を押すとクリップに紐づいたテロップとして登録

   既に作った字幕ファイル (.srt / .vtt) があれば
   [挿入] → [字幕をインポート] で一括登録できます。


━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
  Q3. 色調補正をしたい
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

   1. 対象クリップを選択
   2. Ctrl + G (または [色補正] ボタン)
   3. スライダーで 明るさ・コントラスト・彩度・色温度 などを調整
     - プレビューには GPU シェーダーでリアルタイム反映されます
   4. OK を押すとそのクリップに色補正設定が保存されます


━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
  Q4. 動画を書き出したい (エクスポート)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

   1. タイムラインに編集内容が揃っている事を確認
   2. Ctrl + E (または [出力] ボタン)
   3. ダイアログでコーデック・ビットレート・出力先を設定
      - YouTube アップロード向けなら H.264 / 10 Mbps 程度が一般的
      - 高画質保存なら H.265 / 20 Mbps 以上
   4. エクスポート実行。進捗はプログレスバーで確認
   5. キャンセルも途中で可能


━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
  Q5. 長い動画の中から無音部分を自動で削除したい
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

   1. 対象クリップを選択
   2. [ツール] → [自動ジャンプカット]
   3. 確認ダイアログで Yes
   4. 無音区間が自動検出されてカットポイントが生成されます
   5. 必要に応じて手動で S キーで分割 → 削除


━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
  Q6. J / K / L でプロ風の再生コントロールをしたい
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

   プロの編集ソフト (Premiere Pro / Final Cut など) と同じ操作感です。

   - K:       再生 / 一時停止
   - L:       順方向再生 (連打で 2x → 4x → 8x → 16x)
   - J:       逆方向再生 (連打で -2x → -4x → ...)
   - L → K:   停止
   - J → K:   停止
   - I:       イン点マーク (選択範囲の始まり)
   - O:       アウト点マーク (選択範囲の終わり)


━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
  Q7. 動画にマーカーを付けて YouTube チャプターを作りたい
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

   1. 章の開始位置に再生ヘッドを移動
   2. Ctrl + M でマーカー追加
   3. 名前を入力、種類で [Chapter] を選択
   4. 必要な数だけ繰り返す
   5. [マーカー] → [YouTube チャプターをエクスポート] で
      YouTube 説明欄用のテキストを出力

      例:
        00:00 イントロ
        02:30 本題
        10:15 まとめ


━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
  Q8. 動画の一部をゆっくり再生 (スローモーション) にしたい
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

   方法 A: クリップ全体の速度を変える
     1. 対象クリップを選択
     2. [編集] → [再生速度を設定]
     3. 0.5x (半分速度) などを入力

   方法 B: 速度を徐々に変える (スピードランプ)
     1. 対象クリップを選択
     2. [ツール] → [スピードランプ]
     3. 開始速度・終了速度・イージングを設定


━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
  Q9. プロジェクトを保存して後で再開したい
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

   1. Ctrl + S でプロジェクトを保存 (.veditor 形式)
   2. 次回: [ファイル] → [プロジェクトを開く] で読み込む
   3. 自動バックアップは 2 分おきに ~/.veditor/autosave/ 配下へ保存されます
      クラッシュした場合は次回起動時に復旧ダイアログが出ます


━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
  Q10. ドラッグ & ドロップで動画を追加したい
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

   ウィンドウに MP4 / MKV / MOV / WebM / FLV / PNG / JPG / WAV / MP3 / AAC
   をドラッグ & ドロップで直接追加できます。
   プロジェクトファイル (.veditor) もドロップで開けます。


================================================================================
  付録: トラブルシューティング
================================================================================

  ● 起動時に復旧ダイアログが出る
    前回異常終了した時の自動保存ファイルがあるためです。
    必要なら Yes で復旧、不要なら No で無視して新規作業に進めます。

  ● 動画が落ちる・フリーズする
    対応コーデックでないか、プロキシが必要な 4K/8K 素材の可能性。
    [ツール] → [プロキシ生成] で軽量版を作ってから編集してください。

  ● ログファイル
    クラッシュ時のログは以下に保存されます:
      C:\Users\<ユーザー名>\.veditor\logs\veditor_YYYYMMDD_HHMMSS.log
    スタックトレース付きで記録されるので、バグ報告に添付してください。

  ● 自動保存ファイル
    C:\Users\<ユーザー名>\.veditor\autosave\
    最大 5 世代まで残ります。

  ● プロキシファイル
    C:\Users\<ユーザー名>\.veditor\proxies\
